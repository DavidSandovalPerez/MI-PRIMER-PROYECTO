Si usted está utilizando el GitHub para Mac, simplemente sincronizar su repositorio y verá la nueva sucursal.

### plantillas de diseño
Hemos hecho a mano algunas plantillas guapos para que usted utilice. Seguir adelante y continuar diseños para navegar a través de ellos. Usted puede fácilmente volver a editar su página antes de publicar. Después de publicar su página, usted puede visitar el generador de página y cambiar a otro tema. El contenido de su página se conservará si se mantuvo el formato de rebajas.

### En lugar palillo de la impulsión?
Si prefiere no utilizar el generador automático, empujar una rama llamada 'gh-pages` a su repositorio para crear una página de forma manual. Además de apoyar el contenido HTML regular, GitHub páginas de soporte Jekyll, un blog conscientes sitio estático generador simple, escrito por nuestro propio Tom Preston-Werner. Jekyll hace que sea fácil crear encabezados y pies de página de todo el sitio sin tener que copiarlos a través de cada página. También ofrece soporte de blogs inteligente y otras características avanzadas de plantillas.
